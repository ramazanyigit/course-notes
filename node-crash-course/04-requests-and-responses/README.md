## HTTP Status Codes
### Common Status Codes

- 200: OK
- 301: Resource Moved
- 404: Not found
- 500: Internal server error

### Status Code Ranges
- 100 Range: Informational Responses
- 200 Range: Success Codes
- 300 Range: Codes for redirects
- 400 Range: User or client error codes
- 500 Range: Server error codes
